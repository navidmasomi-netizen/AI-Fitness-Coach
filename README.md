# AI-Fitness-Coach

AI-Fitness-Coach is a mobile-first fitness coaching application.

This is not simply a workout logging application.

The long-term vision is to build an intelligent fitness coach that helps users with:

- Workout programming
- Workout execution
- Progression tracking
- Nutrition
- Supplement reminders
- Recovery
- Daily coaching
- AI assistance

---

## Tech Stack

### Mobile

- React Native
- Expo
- TypeScript
- React Query
- Zustand

### Backend

- Express
- Prisma
- PostgreSQL
- JWT Authentication
