# PROJECT KNOWLEDGE — AI-Fitness-Coach

## 1. Product Vision

AI-Fitness-Coach is a mobile-first fitness coaching application.

It is not just a workout logger.

The long-term vision is to build a full daily AI fitness coach that helps users with:

- workout planning
- workout execution
- progressive overload
- workout history
- nutrition guidance
- supplement reminders
- recovery tracking
- habit building
- daily coaching
- AI-assisted decision making

The product should eventually become a daily companion for athletes, not only an app used during workouts.

---

## 2. Product Philosophy

The application should guide the user instead of forcing the user to think.

Core principle:

> Do not make the user decide what to do next. Show them clearly.

The user should be able to:

1. Open the app
2. Understand today’s workout immediately
3. Start the session
4. Log sets quickly
5. Finish the workout
6. See progress and next recommendations

The app should feel like a coach, not a spreadsheet.

---

## 3. Current Product Direction

The current product is focused on:

- workout execution
- guided workout start
- set logging
- rest timer
- workout completion
- workout history
- progression recommendations
- lightweight habit cues

Future direction:

- first-time clarity onboarding
- fitness profile onboarding
- personalized program generation
- nutrition planning
- supplement reminders
- daily coaching dashboard
- AI coach layer
